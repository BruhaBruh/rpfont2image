# Resourcepack font to \`symbol.png\`

## Download

1. Move to release page
2. Download file for your architecture
3. Enjoy!


## Usage

Default variables:
- `input` `resourcepack`
- `output` `output`

### Linux

```bash
rpfont2image -input resourcepack -output output
```

### Windows

```shell
rpfont2image.exe -input resourcepack -output output
```
